# User Consent Statement

This prototype is created for ICT105 learning purposes. We ask users to test the **Peer Tutoring Booking Platform** prototype and provide feedback. The information collected will be used only to improve the project prototype. Users should not provide sensitive personal information. Participation is voluntary.

## Data Collected

- **Name/nickname or anonymous ID:** e.g. "Tester T04" (as used in `test-user-notes.md`)
- **Feedback score:** usability rating from the feedback form (`data/feedback-form-results.csv`)
- **Comments about usability:** free-text comments on the booking and search flow
- **Task completion result:** whether the tester completed the search → book → status-check tasks (`data/experiment-results.csv`)

## Data Use

The collected data will be used only for:
- Class project analysis
- Prototype improvement
- Final project reporting for ICT105

The data will **not** be shared outside the project team or used for purposes beyond course assessment.
