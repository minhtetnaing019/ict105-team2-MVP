## IP Summary
The prototype uses a small number of external assets: Font Awesome icons, Google Fonts, and the Bootstrap CSS framework, all under permissive open licenses (or free tiers) suitable for a class project. No proprietary code, paid datasets, or third-party APIs are used. One early wireframe icon with an untracked source was identified during this review and will be replaced before the final prototype build (see Risk Register R-03). All other assets are either team-created (placeholder avatars) or course-provided materials, both low risk.
